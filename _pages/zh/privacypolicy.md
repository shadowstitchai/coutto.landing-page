# 隐私政策

欢迎使用 **纽扣纸样**（Coutto Pattern，以下简称「我们」或「本应用」）。我们重视您的隐私，并致力于保护您的个人信息。本隐私政策说明我们在您使用本移动应用程序（iOS 与 Android）、网站 [coutto.app](https://coutto.app) 及相关服务时，如何收集、使用、存储、共享和保护您的信息。

使用本应用即表示您已阅读并理解本政策。若您不同意本政策，请停止使用我们的服务。

<br>

## 1. 我们收集的信息

### 1.1 您主动提供的信息

- **账户信息**：注册或登录时提供的邮箱地址、用户名、密码（加密存储）及头像等资料
- **用户内容**：您导入、创建或上传的纸样文件（PDF、SVG、DXF、PLT、PNG、JPEG、WebP 等）、标注、分类与偏好设置
- **社区内容**：您在缝纫社区中发布的帖子、评论、图片及其他互动内容（如适用）
- **反馈与客服**：您通过邮件或应用内渠道提交的问题、建议或投诉

### 1.2 自动收集的信息

- **设备信息**：设备型号、操作系统版本、唯一设备标识符、屏幕分辨率、语言与地区设置
- **应用使用数据**：功能使用频率、崩溃日志、性能数据、会话时长、点击与浏览行为
- **网络信息**：IP 地址、网络类型（Wi-Fi / 蜂窝数据）
- **诊断信息**：错误报告、崩溃堆栈，用于排查问题并提升稳定性

### 1.3 来自第三方的信息

- **应用商店**：Apple App Store、Google Play 提供的购买记录、订阅状态及匿名标识（我们不直接收集您的支付卡信息）
- **统计分析服务**：第三方 SDK 提供的聚合使用数据（详见第 4 节）
- **云同步服务**：用于账户登录与数据同步的后端服务返回的会话与存储信息

<br>

## 2. 我们如何使用您的信息

我们将收集的信息用于以下目的：

| 用途 | 说明 |
| :--- | :--- |
| 提供核心功能 | 纸样查看、投影裁切、拼接、打印、存储与管理 |
| 账户与同步 | 创建和管理账户，在您的设备间同步纸样与设置 |
| 改善产品体验 | 分析使用趋势，优化界面与功能 |
| 保障安全 | 检测滥用、欺诈及未经授权的访问 |
| 客户支持 | 回复您的咨询与投诉 |
| 法律合规 | 履行法律法规规定的义务 |
| 推送通知 | 在您授权的情况下，发送功能更新、订阅提醒等通知 |

## 4. 第三方 SDK 及共享

为保障 App 相关功能的实现与应用安全稳定运行，我们接入由第三方提供的软件开发包（SDK）及云服务。我们会对合作方进行安全评估，并要求其依法保护您的信息。

### 4.1 第三方 SDK 清单

| SDK 名称 | 服务类型 | 收集的个人信息类型 | 隐私政策链接 |
| :--- | :--- | :--- | :--- |
| 友盟+ SDK | 移动应用数据统计与分析 | 设备信息（IMEI / Android ID / IDFA / OPENUDID / GUID / SIM 卡 IMSI）、网络信息、应用使用行为；在授权情况下可能收集粗略地理位置 | [https://www.umeng.com/page/policy](https://www.umeng.com/page/policy) |
| Apple 框架（iOS） | 应用分发、App 内购买、崩溃报告（如启用） | 设备标识符、购买记录、崩溃日志 | [https://www.apple.com/legal/privacy/](https://www.apple.com/legal/privacy/) |
| Google Play 服务（Android） | 应用分发、App 内购买、崩溃报告（如启用） | 设备标识符、购买记录、崩溃日志 | [https://policies.google.com/privacy](https://policies.google.com/privacy) |

### 4.2 其他第三方服务

| 服务提供方 | 用途 |
| :--- | :--- |
| 云存储与同步服务 | 账户认证、纸样与用户数据的云端存储及多设备同步 |
| 应用商店（Apple / Google） | 应用分发、订阅与 App 内购买结算 |

除上述情形外，我们不会向第三方出售您的个人信息。在以下情况下，我们可能共享信息：

- 经您明确同意
- 为履行法律义务或响应政府机关的合法要求
- 为保护我们、用户或公众的权利、财产与安全
- 在合并、收购或资产转让中，依法向继受方转移（我们会提前通知您）

## 5. 数据存储与安全

### 5.1 存储地点与期限

- 您的数据主要存储于中华人民共和国境内及/或我们云服务提供商所在的数据中心
- 我们仅在实现本政策所述目的所必需的期限内保留您的个人信息；账户注销后，我们将在合理期限内删除或匿名化处理您的数据，法律法规另有规定的除外

### 5.2 安全措施

我们采取行业标准的安全措施保护您的信息，包括但不限于：

- 传输层加密（HTTPS / TLS）
- 密码加密存储
- 访问权限控制与审计
- 定期安全评估

尽管我们尽力保护您的信息，但互联网传输与电子存储无法保证绝对安全。请您妥善保管账户凭证。

<br>

## 6. 您的权利

### 6.1 一般权利

根据适用的数据保护法律（包括《个人信息保护法》），您享有以下权利：

1. **查阅与复制**：请求获取我们持有的您的个人信息副本
2. **更正**：请求更正不准确或不完整的信息
3. **删除**：在符合法定条件时，请求删除您的个人信息
4. **撤回同意**：撤回您此前给予的同意（不影响撤回前基于同意的处理合法性）
5. **注销账户**：通过应用内设置或联系我们注销账户
6. **投诉**：向有管辖权的监管机构投诉

### 6.2 欧洲经济区（EEA）用户 — GDPR

若您位于欧洲经济区，您还享有：限制处理、数据可携带、反对基于合法利益的处理，以及向欧盟成员国监管机构投诉的权利。我们处理您数据的法律依据包括：履行合同、合法利益、您的同意，以及法律义务。

### 6.3 加利福尼亚州居民 — CCPA / CPRA

若您是加州居民，您有权：

- 知悉我们收集、使用、共享和出售的个人信息类别
- 请求删除我们持有的您的个人信息（法定例外除外）
- 更正不准确的个人信息
- 选择退出个人信息的「出售」或「共享」（**我们不会出售您的个人信息**）
- 不因行使上述权利而受到歧视

如需行使上述权利，请通过第 10 节联系方式与我们联系。我们将在法定期限内回复。

<br>

## 7. 未成年人保护

本应用主要面向年满 **14 周岁** 的用户。我们不会故意收集 14 周岁以下儿童的个人信息。若您是未成年人的监护人，发现我们未经同意收集了儿童信息，请联系我们，我们将尽快删除。

<br>

## 8. 跨境传输

若您的信息需传输至中华人民共和国境外，我们将依据适用法律采取必要措施（如标准合同条款、安全评估等），确保您的信息得到与本政策同等水平的保护。

<br>

## 9. 政策变更

我们可能不时更新本隐私政策。重大变更时，我们将通过应用内通知、网站公告或其他适当方式告知您。更新后的政策自公布之日起生效；若您继续使用本应用，即视为接受修订后的政策。
